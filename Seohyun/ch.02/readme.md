# 변수

🛩**변수의 타입**🛩
* 기본형
  * boolean, char, short, int, byte(이진데이터 다룰때) 등등
* 참조형
  * String, System 등

🛩**명명규칙**🛩
1. 대소문자 구분됨
    > True랑 true는 다르다. (true는 예약어라 사용 불가능)
2. 숫자로 시작하면 안됨
3. 특수문자는 '_'랑 '$'만 허용
4. 클래스 이름 첫 글자는 항상 대문자
5. 변수, 메서드 이름 첫 글자는 항상 소문자
6. 여러 단어 이름은 단어 첫 글자 모두 대문자
   > 단어는 '_'로 구분함
7. 상수의 이름은 대문자
